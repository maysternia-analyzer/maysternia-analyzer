# Майстерня скілів — аналізатор записів

Локальне веб-застосування для аналізу пробних занять і продажів менеджерів онлайн-школи "Майстерня скілів".

## Запуск

1. Відкрийте папку `maysternia-analyzer`.
2. Скопіюйте `.env.example` у `.env`.
3. Заповніть `OPENAI_API_KEY` і `ANTHROPIC_API_KEY`.
4. Встановіть залежності:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

5. Запустіть застосунок:

```bash
python app.py
```

Якщо на Mac команда `python` не налаштована, використайте:

```bash
python3 app.py
```

Після запуску відкрийте `http://localhost:5050`.

## Що є всередині

- Завантаження Zoom-записів у форматах `mp4`, `m4a`, `mp3`, `wav`, `webm`, `mov`.
- Транскрипція через OpenAI Whisper API, модель `whisper-1`.
- Аналіз через Anthropic Claude, модель з `ANTHROPIC_MODEL` у `.env`.
- Два чек-лісти: пробне заняття тренера і продаж менеджера.
- Дашборд з фільтрами, кольоровими індикаторами, таблицями і деталями запису.
- Детальна сторінка з плеєром, транскрипцією, AI-аналізом, коментарем керівника і повторним аналізом.
- Локальне збереження в SQLite-файлі `data.db`.

Під час першого входу створіть адміністратора на сторінці налаштування. У полі секретного ключа введіть значення `SECRET_KEY` з `.env`.
